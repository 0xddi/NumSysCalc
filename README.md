# NumSysCalc
Небольшое C# приложение для работы с числами в разных системах счисления и представлению их в ЭВМ. Версия на ассембли по ссылке: https://github.com/0xddi/NumSysAsm
# Инструкция
Программа работает через принятие определённых команд от пользователя. Для получения их списка просто напишите 'help' или '?'.
# Возможности и особенности:
- Перевод из любой системы счисления в любую другую с.с. в рамках используемого алфавита (напишите 'alphabet' для его вывода);
- Основные операции над числами: умножение и сложение;
- Несколько операций за один ввод с помощью 'псевдосинтаксиса' (см. вывод команды 'help' для подробностей);
- Представление данных в ЭВМ: прямой код, обратный код, дополнительный код, представление вещественных чисел;
- Все вышеобозначенные пункты снабжены подробными комментариями, которые будут выведены во время непосредственного выполнения определённых команд.
